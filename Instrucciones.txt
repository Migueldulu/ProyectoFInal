Para compilar la LibSinCompilar:

    - Me voy a la carpeta que contiene todos los archivos (cpp) y ejecuto:
            $NDK = "C:/Program Files/Unity/Hub/Editor/6000.2.5f1/Editor/Data/PlaybackEngines/AndroidPlayer/NDK"
            cmake -S . -B build-arm64 -G "Ninja" -D CMAKE_BUILD_TYPE=Release -D CMAKE_TOOLCHAIN_FILE="$NDK/build/cmake/android.toolchain.cmake" -D ANDROID_ABI=arm64-v8a -D ANDROID_PLATFORM=24 -D ANDROID_STL=c++_static -D ANDROID_ARM_NEON=ON
            cmake --build build-arm64 --config Release

    - Se deberia haber creado en build-arm64 un archivo llamado libreriaTelemetria.so (yo lo copio a LibreriaTelemetria/AARTelemetria/src/main/jniLibs/arm64-v8a para crear el aar)
